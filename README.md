# lab2-task1-gr12b-skxfn

## Overview

Задание 1 лабораторной работы №2. Консольное приложение на языке C. Вариант 13.

Дан массив натуральных чисел размерности n. Определить количество чисел, десятичная запись которых не содержит одинаковых цифр.

## Author

Ласицкий Никита, группа 12б

## Usage

```bash
gcc -o labrabota2-1 src/main.c
./labrabota2-1 5 12345 11223 9876 1001 54321
```

## Building

Данные передаются через аргументы командной строки: первый аргумент — количество чисел, далее — сами числа.
